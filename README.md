## 🎯 Mục tiêu

Rèn luyện kỹ năng tạo, đổi tên, sao chép, di chuyển, xóa tệp tin và thư mục một cách thành thạo trên hệ điều hành **Windows**.

## 📋 Nội dung thực hành

Bài thực hành bao gồm **12 bước** được minh họa chi tiết bằng ảnh chụp màn hình:

| # | Thao tác | Mô tả |
|---|----------|-------|
| 1 | Mở File Explorer | `Win + E` hoặc nhấp biểu tượng trên taskbar |
| 2 | Truy cập ổ đĩa | Chọn This PC → ổ đĩa D: hoặc E: |
| 3 | Tạo thư mục mới | Chuột phải → New → Folder |
| 4 | Vào thư mục | Nhấp đúp để mở thư mục |
| 5 | Tạo tệp tin văn bản | New → Text Document |
| 6 | Đổi tên tệp tin | Chuột phải → Rename |
| 7 | Tạo thư mục con | Tạo folder con bên trong |
| 8 | Copy & Paste | Sao chép tệp vào thư mục khác |
| 9 | Cut & Paste | Di chuyển tệp sang vị trí mới |
| 10 | Xóa tệp tin | Delete → Recycle Bin |
| 11 | Xóa vĩnh viễn | Shift + Delete |
| 12 | Khôi phục từ Thùng rác | Restore từ Recycle Bin |

## 🚀 Xem báo cáo

Mở file `index.html` trong trình duyệt để xem báo cáo trực quan với giao diện web hiện đại.

```
📦 Intro-to-Digital-Tech
├── 📄 index.html        ← Trang web báo cáo chính

├── 📁 images/           ← Ảnh chụp minh họa (22 ảnh)
└── 📘 README.md         ← File này
```
